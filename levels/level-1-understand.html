1.1 — Tại Sao Agent?

Context Engineering

Output Quality = f(Model × Context × Constraints). Cùng 1 model, nhưng: Không context → 40% · Prompt tốt → 60% · System instructions → 75% · Agent system → 90%+. Càng nhiều context phù hợp → output càng chính xác.

1.2 — Gemini Gems

Gem là gì?

Gem = một version Gemini được customize với system instructions cố định. Khi dùng Gem, AI luôn tuân theo instructions đó mà không cần nhắc lại. Concept giống nhau trên mọi platform — hiểu 1 → dùng được tất cả.

Template: Tạo Gem

# Role
Bạn là copywriter chuyên nghiệp cho [TÊN CÔNG TY].

# Brand Voice
- Tone: [Chuyên nghiệp / Thân thiện / Mạnh mẽ]
- Audience: [Mô tả khách hàng mục tiêu]
- Tránh: [Từ ngữ nào không dùng]
- Luôn: [Đặc điểm nào phải có]

# Format Rules
- Tiêu đề: Max 8 từ, có action verb
- Body: Max 3 đoạn, mỗi đoạn max 3 câu
- CTA: Luôn kết thúc bằng 1 call-to-action

# Examples
[2-3 ví dụ output mẫu — PHẦN QUAN TRỌNG NHẤT]
Best Practice

Few-shot examples là yếu tố tạo chất lượng lớn nhất. AI bắt chước style từ examples tốt hơn bất kỳ instruction nào.

1.3 — 5 Prompt Patterns

Pattern 1: Role → Task → Constraints → Format

You are a [ROLE] with expertise in [DOMAIN].
Your task is to [SPECIFIC TASK].
Constraints: [BOUNDARIES]
Output format: [STRUCTURE]

Pattern 2: Chain of Thought (CoT)

Think step by step:
1. First, analyze [X]
2. Then, consider [Y]
3. Finally, recommend [Z]
Show your reasoning at each step.

Pattern 3: Few-Shot Learning

Here are examples of what I want:
Input: [example 1] → Output: [example 1]
Input: [example 2] → Output: [example 2]
Now do this: Input: [new input] → Output: ?

Pattern 4: Negative Prompting

DO NOT:
- Use generic phrases like "in today's world"
- Write more than 3 paragraphs
- Include jargon without explanation

Pattern 5: Iterative Refinement

[Prompt ban đầu]
→ "Kết quả chưa đúng vì [lý do]. Hãy chỉnh [phần cụ thể]"
→ "Tốt hơn, nhưng [X] cần thay đổi. Hãy [Y]"
