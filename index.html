Vấn Đề

Tại Sao Workshop Này Tồn Tại

95% người dùng AI vẫn ở cấp "hỏi - chờ trả lời". Chưa khai thác khả năng agentic — nơi AI chủ động, tuân theo quy trình, và phối hợp nhiều chuyên gia cùng lúc.

Cấp ĐộNgười DùngKết Quả
Lv.1 — Chat95%Copypaste, chất lượng lồi lõm, không nhất quán
Lv.2 — Custom Instructions~4%Output tốt hơn nhưng vẫn manual mỗi lần
Lv.3 — Agent System<1%AI tuân thủ quy trình, tự routing, chất lượng ổn định

Khung Khái Niệm

Prompt
Lệnh 1 lần, không lưu trữ
Đơn giản nhất
"Dịch đoạn này sang tiếng Anh"
Skill
Kiến thức chuyên sâu, tái sử dụng
Reusable
"Biết cách viết React component chuẩn"
Workflow
Quy trình nhiều bước, có thứ tự
Repeatable
"Lint → Test → Build → Deploy"
Agent
Vai trò + Quyết định + Phối hợp
Autonomous
"Frontend specialist tự chọn skill, routing task"
Phép tương đồng — Nhà hàng: Prompt = order 1 món · Skill = công thức nấu · Workflow = quy trình từ order → ra bàn · Agent = bếp trưởng biết chọn công thức, phối hợp bếp phụ.
